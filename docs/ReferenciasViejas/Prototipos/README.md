# Prototipos CSH — Tres propuestas

Tres prototipos en HTML y CSS para la página web del Computer Science Hub, según el plan de propuestas creativas. Todos aplican la guía común (minimalismo editorial, paleta hueso/papel, grid 12 columnas, tipografía serif/sans).

## Cómo ver los prototipos

Abre en el navegador el archivo `index.html` de cada carpeta:

- **propuesta-1-ecosistema** — Ecosistema en evolución (scroll narrativo, recorrido, misión, valores, CTA).
- **propuesta-2-hub-vivo** — Hub vivo (hero, Actividad reciente, pilares, metas y plazos, participar).
- **propuesta-3-ingenieria-rostro** — Ingeniería con rostro (comunidad, valores con énfasis, nuestro camino, testimonios, únete).
- **anthropic-prototipo** — Réplica de referencia del sitio [anthropic.com](https://www.anthropic.com/): minimalismo editorial, hero con subrayado en palabra clave, sección "Latest releases" y statement con enlaces temáticos. Sirve como referencia visual para la guía de diseño del proyecto CSH.

No se requiere servidor; los estilos y fuentes (Google Fonts) se cargan por enlace.
